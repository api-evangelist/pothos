# Pothos Plans

Pothos GraphQL is an open-source library distributed under the ISC license. There are no commercial pricing tiers or subscription plans.

## Open Source (Free)

- Full access to `@pothos/core` and all official plugins
- Available via npm: `npm install @pothos/core`
- Zero runtime overhead; GraphQL is the only peer dependency
- Unlimited use in personal, commercial, and enterprise projects

## Sponsorship

Pothos is community-funded. Sponsors can support development via the GitHub Sponsors page. See https://pothos-graphql.dev/docs/sponsors for details.

## Enterprise Usage

Pothos is used in production at companies including Airbnb and Netflix. No commercial support tier is offered; enterprise users rely on the open-source community and GitHub Discussions.
